# K3s Deterministic Networking Research

Research project analyzing deterministic networking performance in K3s Kubernetes clusters. Comparing Flannel vs Cilium CNI plugins for latency, jitter, and packet loss metrics to enable real-time applications in cloud-native environments.

## Features
- K3s cluster performance benchmarking
- Flannel VXLAN vs Cilium eBPF comparison  
- QoS metrics analysis for DetNet requirements
- Automated test suites and result collection

## Tech Stack
K3s • Flannel • Cilium • eBPF • Kubernetes • iperf3 • Go
